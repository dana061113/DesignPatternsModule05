# DesignPatternsModule05

Бұл Java жобасы – **Модуль 05: Порождаушы паттерндер**.  
Жоба үш негізгі паттернді қамтиды: **Singleton (Одиночка), Builder (Строитель), Prototype (Прототип)**.  

---

## Паттерндердің сипаттамасы

### 1️⃣ Singleton (Одиночка)
- Бір ғана объектіні құруға мүмкіндік береді.
- Глобалды түрде қолжетімділік қамтамасыз етеді.
- `ConfigurationManager` класы арқылы қолданылған.

### 2️⃣ Builder (Строитель)
- Күрделі объектіні біртіндеп құруға мүмкіндік береді.
- Мысалы, түрлі форматтағы есептер (`TextReport`, `HtmlReport`) жасалады.
- `IReportBuilder`, `TextReportBuilder`, `HtmlReportBuilder`, `ReportDirector` арқылы жүзеге асырылған.

### 3️⃣ Prototype (Прототип)
- Объектілерді шаблон арқылы тез және тиімді көшіруге мүмкіндік береді.
- Мысалы, интернет-дүкендегі тапсырыстарды (`Order`) және тауарларды (`Product`) клонирлеу.
- `ICloneable`, `Product`, `Order` арқылы жүзеге асырылған.

---

## Жобаны іске қосу

1. GitHub-та репозиторияны ашыңыз.  
2. `Main.java` файлын кез келген онлайн Java редакторында немесе IDE-де іске қосыңыз.  
3. Бағдарлама үш паттернді тестілейді:  
   - Singleton: бір ғана конфигурация объектісінің тексерісі  
   - Builder: тексттік және HTML есептерді құру  
   - Prototype: тапсырысты клонирлеу  

---

## Мақсаты

- Singleton арқылы конфигурацияларды бір ғана объект арқылы басқаруды үйрену.  
- Builder арқылы есептерді немесе күрделі объектілерді икемді түрде құру.  
- Prototype арқылы объектілерді клонирлеу және өзгерту тәжірибесін алу.
